# Cowsay_Holberton_School
Holberton School logo for cowsay

## Install cowsay
```$ sudo apt-get install cowsay```

## Add the holberton file to the cows folder
Add file holberton.cow to the repo "/usr/share/cowsay/cows/"

## Run
```$ cowsay -f holberton ':(){ :|:& };:'```
